# is-isogram

Test whether a word or phrase is an [isogram](https://en.wikipedia.org/wiki/Isogram) (a word or phrase without a repeating letter).

## Installation

```bash
$ npm install is-isogram --save
```

## Usage

```js
var isIsogram = require('is-isogram');

isIsogram('sandwich');
// -> true

isIsogram('wedding');
// -> false
```
